# 📋 Plano de Ação (Task Plan) - CzTechnology Landing Page

## 🎯 Objetivo
Criar uma landing page profissional, estratégica e de alta conversão para a CzTechnology, focada em automação de atendimento e desenvolvimento web.

## 🚀 Fases do Projeto

### Fase 0: Inicialização (Protocolo 0)
- [x] Criar arquivos de memória (`task_plan.md`, `findings.md`, `progress.md`, `gemini.md`).
- [ ] Definir constituição técnica e visual da marca.

### Fase 1: Visão Estratégica & Copywriting (V & L)
- [x] Definir público-alvo e dores.
- [x] Mapear os links de destino e CTAs.
- [x] Escrever a copy de todas as seções (Hero, Dor, Solução, Serviços, etc.).

### Fase 2: Arquitetura e Estrutura HTML (A)
- [x] Configurar ambiente Vanilla HTML/CSS/JS.
- [x] Construir a estrutura semântica de cada seção da landing page.

### Fase 3: Design System e Estilo (E)
- [x] Implementar a paleta de cores (Dark Mode/Neon Tech).
- [x] Configurar tipografia (Ex: Inter, Roboto).
- [x] Aplicar estilos globais e componentes (Botões, Cards).
- [ ] Inserir animações GSAP ou CSS puro para dar um toque "Wow".

### Fase 4: Gatilho e Finalização (G)
- [ ] Testes de Responsividade (Mobile, Tablet, Desktop).
- [ ] Otimização de Performance e SEO básico.
- [ ] Checklist final de conversão (CTAs funcionais, contraste, hierarquia).

## ✅ Checklist de Qualidade (V.L.A.E.G.)
- A página tem um objetivo comercial claro?
- Os CTAs se repetem ao longo da página?
- O visual transmite profissionalismo e alta tecnologia?
- A estrutura segue a ordem lógica de persuasão (Atenção -> Dor -> Solução -> Ação)?
