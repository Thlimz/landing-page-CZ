## Plano de Ação

1.  **Fase 0: Blueprint (Protocolo 0)**
    *   Criar os 4 arquivos de memória.
    *   Analisar as imagens de inspiração para extrair padrões de design.
    *   Preencher a Visão Estratégica (V) baseada no perfil da CzTechnology.

2.  **Fase 1: Estrutura e Copy (A - Arquitetura)**
    *   Definir a hierarquia das seções (Hero, Dor, Solução, etc.).
    *   Escrever a copy persuasiva para cada seção.

3.  **Fase 2: Design System (E - Estilo)**
    *   Configurar `index.css` com variáveis de CSS modernas.
    *   Implementar o sistema de grid e espaçamento.

4.  **Fase 3: Desenvolvimento (Front-end)**
    *   Construir a estrutura semântica HTML.
    *   Aplicar o design e animações GSAP (se necessário para o toque "wow").

5.  **Fase 4: Validação (G - Gatilho)**
    *   Testar responsividade e performance.
    *   SEO e checklists finais.
